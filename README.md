# Ascension Client documentation (GitHub-friendly)

This repository combines:

1) **Interface wiki** (generated from the shipped `Interface/` package)
2) **Runtime scan results** (captured in-game and annotated)

## Start here

- [docs/README.md](docs/README.md)

## Runtime scan (Markdown)

The `docs/RuntimeScan/` docs are the human-readable, GitHub-friendly write-ups of what was observed at runtime.

The `Bronzebeard Scan/` folder contains the **v2 Bronzebeard Scan exports** (HTML indexes + raw Markdown/JSON dumps)

- [Bronzebeard Scan - API index (HTML)](Bronzebeard%20Scan/index_v2.html)
- [Bronzebeard Scan - Globals/UI objects index (HTML)](Bronzebeard%20Scan/globals_index_v2.html)
