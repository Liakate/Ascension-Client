# Runtime scan

This section contains **in-game runtime scan results** for the Ascension client.

These documents complement the Interface-derived wiki pages:

- **Interface docs** show what the shipped UI *calls/uses*.
- **Runtime scan** shows what the live client *exposes at runtime* (including APIs not referenced by the shipped UI).

## Files

- [API_Documentation_RuntimeScan.md](API_Documentation_RuntimeScan.md) — high-level API documentation extracted from runtime evidence.
- [API_Runtime_Objects.md](API_Runtime_Objects.md) — runtime object inventory.
- [Global_UI_Objects.md](Global_UI_Objects.md) — `_G` UI objects inventory (large table).
- [Object_Graph_Fields.md](Object_Graph_Fields.md) — common object graph fields.
- [EvidenceIndex.json](EvidenceIndex.json) — raw evidence index.
- [EvidenceIndex_VerificationReport.md](EvidenceIndex_VerificationReport.md) — verification report.
- [GlobalsIndex.json](GlobalsIndex.json) — raw globals index.
